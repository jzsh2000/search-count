search-count
============

Usage
-----

```
usage: search-count.py [-h] -i INPUT [-o OUTPUT] [-s {baidu}]

Search & Count

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        input file
  -o OUTPUT, --output OUTPUT
                        output file
  -s {baidu}, --source {baidu}
                        search engine
```
